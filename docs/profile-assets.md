# Profile Assets

This repo stores the interim GitHub organization avatar for
`Great-Falls-Tool-Bus`.

## GitHub avatar

- Prepared file: `docs/assets/github-avatar.png`
- Size: 512x512 PNG
- Source: Great Falls Pride - Lewiston Auburn image,
  `https://images.squarespace-cdn.com/content/v1/664cc2088b45af2a1644bed2/eb77e5c7-3b44-4af3-847b-943b507a5f8d/Untitled%2Bdesign%2B%2814%29.png`
- Intended use: interim GitHub organization profile image for
  `https://github.com/Great-Falls-Tool-Bus`

GitHub's supported REST and GraphQL APIs can update organization profile fields,
but do not expose an organization avatar upload mutation. Upload the prepared
PNG through the organization Settings profile image control.
